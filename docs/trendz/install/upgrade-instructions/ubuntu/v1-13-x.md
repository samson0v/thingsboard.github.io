{% capture difference %}
**NOTE:**
<br>
After upgrading it is required to update Trendz widget bundle in ThingsBoard. You can find detailed instructions how to do that via Trendz UI in [Trendz Widget Bundle upgrade](/docs/trendz/post-installation-steps/#trendz-widget-bundle) article.

#### Trendz Analytics package download

{% include resources.liquid version="1.13.2" kind="ubuntu-download" trendz="true" %}  

#### Trendz Analytics service upgrade

* Install Trendz Analytics service 

{% include resources.liquid version="1.13.2" kind="ubuntu-installation" trendz="true" %} 

**NOTE:** Package installer will ask you to merge your trendz configuration. It is preferred to use **merge option** to make sure that all your previous parameters will not be overwritten.

Execute regular upgrade script:

```bash
sudo /usr/share/trendz/bin/install/upgrade.sh --fromVersion=1.13.1
```
{: .copy-code}

#### Start the service

```bash
sudo service trendz start
```
{: .copy-code}
